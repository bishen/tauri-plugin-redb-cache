## Default Permission

Default permissions for redb-cache plugin - allows all cache operations

#### This default permission set includes the following:

- `allow-cache-get`
- `allow-cache-set`
- `allow-cache-remove`
- `allow-cache-clear`
- `allow-cache-clean-expired`
- `allow-cache-info`
- `allow-cache-list`
- `allow-image-cache-get`
- `allow-image-cache-set`
- `allow-image-cache-remove`
- `allow-image-cache-clean-expired`
- `allow-image-cache-clear`
- `allow-image-cache-info`
- `allow-image-cache-list`
- `allow-cache-ns-get`
- `allow-cache-ns-set`
- `allow-cache-ns-remove`
- `allow-cache-ns-get-batch`
- `allow-cache-ns-set-batch`
- `allow-cache-ns-get-by-prefix`
- `allow-cache-ns-remove-by-prefix`
- `allow-cache-ns-clear`
